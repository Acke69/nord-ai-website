# NORD.AI — Webb Intelligence Studio

AI-driven web agency landing page built with vanilla HTML, CSS and JavaScript.

## Features

- Reactive parallax ring animation (canvas) responding to mouse movement
- Alternating dark/light sections with bento grid layout
- Horizontal scroll process strip
- Interactive FAQ accordion
- Custom cursor with lag ring
- Scroll-reveal animations
- Fully in Swedish

## Stack

Pure HTML · CSS · Vanilla JS · Space Grotesk + Space Mono (Google Fonts)

## Live preview

Open `index.html` directly in your browser, or deploy via GitHub Pages.
