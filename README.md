# NORD.AI — Webb Intelligence Studio

AI-driven web agency landing page built with vanilla HTML, CSS and JavaScript.

## Filer

| Fil | Beskrivning |
|-----|-------------|
| `index.html` | Landningssida (svenska) |
| `kontakt.html` | Kontaktsida med bokningsformulär |

## Stack

Pure HTML · CSS · Vanilla JS · Three.js (globe) · Space Grotesk + Space Mono (Google Fonts)

## Features

- Interaktiv 3D-glob med Three.js (drag, dubbelklick-zoom, pulserande stadspunkter)
- Karusell med 5 processteg (4 synliga på desktop)
- Alternerade mörka/ljusa sektioner med bento-grid
- Interaktiv FAQ-accordion
- Custom cursor med lagring
- Scroll-reveal-animationer
- Kontaktformulär kopplat till Formspree

## Underhåll

Detta projekt underhålls via **Claude AI + GitHub API**.  
Ändringar pushas direkt till `main`-branchen.

> ⚠️ **Om du använder Lovable:** Redigera **inte** `index.html` eller `kontakt.html` direkt i Lovable — det kan skriva över Claude-ändringar. Använd Lovable enbart för att förhandsgranska.
